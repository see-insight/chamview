Python2.7, PIL 1.1.7, and OpenCV 2.2 are the dependencies for cvmac.

Cvmac is the main program with all of the functionality, you can use your own list of points by command line argument when starting cvmac.  Champts.txt is the default list of points.  Points.txt is a file that allows for custom point functionality.  Cvmac works on both mac os and windows.   

sc.py will count chameleon steps given a text file.
